# scripts

## Install

### npm (recommended)

```sh
npm install -g @blackglory/scripts
# or
yarn global add @blackglory/scripts
```

### Linux with Zsh

```sh
git clone https://github.com/BlackGlory/scripts.git
./scripts/install-scripts.sh
```

### Windows with Powershell

```ps
git clone https://github.com/BlackGlory/scripts.git
./scripts/install-scripts.ps1
```
